# Credit Card Financial Dashboard

A simple and interactive Power BI dashboard built using credit card customer and transaction data.
This project gives clear insights into spending patterns, customer segments, and overall financial performance.

- Project Overview

✔ This dashboard helps analyze:
✔ Total transaction amount & counts
✔ Customer demographics
✔ Monthly & weekly spending trends
✔ Revenue insights
✔ Customer behavior and usage patterns
✔ The goal is to make financial analysis easy, visual, and useful for business decisions.

- Tech Used

✔ Power BI – Dashboard and visualizations
✔ SQL – Data cleaning and preprocessing
✔ DAX – Measures and calculations
✔ CSV Files – Raw datasets


- Features

✔ Clean and simple design
✔ Useful for finance teams
✔ Easy to modify and extend
✔ Beginner-friendly structure
